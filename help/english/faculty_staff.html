2. Dashboards ¶

This module will give the user a clear detailed information about his\her courses along with charts, it also shows tasks and surveys forwarded to the user to be done.

  1. Access the dashboard by clicking on” dashboard” button from the main menu.

  2. Click on” personal” button to access personalinformation, and view tasks and survey forwarded.

  3. Personal page shall appear for the user, along with tasks and surveys (if there is any).

  4. Click on” Take survey” button to fill up survey assigned.

  5. Fill up the survey, when done click on” finish”.

  6. Manage the tasks from tasks tab, assign a task to another user by clicking on ”add new” button.

  7. Fill up required information, when done click on ”Send” button.
  8. Click on ”general” to see general information about the institution.

3. Accreditation ¶

Through this module the user will be able to control the accreditation process assigned to him\her.

  1. Access the module by clicking on ”Accreditation” button, then choose the sub-category required.

  2. Click on ”manage” to access assigned forms.

  3. There are function buttons to help the user managing the forms.
    1. Set a due date.
    2. Assign a user to fill or manage the forms.
      • Click on the ”add user” button, then choose whattype of user you want to add.

      • find the required user by searching the users name, select the users name then click on add.

    3. Edit forms button.
      • Click on edit forms button, then fill up or edit the form as needed, when done click on ”Save Changes” to save the form without marking it as finished (to be finished later), or click on ”Save and finish” to save the form and mark it as finished.

      • to retrieve a previous history for the form, click on the ”history” button top right of the form

      • the user will be able to see both forms, the current form and the chosen form from history function, every change that has been made on the form will be reflected as a log along with the date and time.

      • Change or keep the needed information, then click on ”Save Changes”.
    4. View the form.
      • the user will be able to view the current form by clicking on ”View” button.

    5. Send to reviewer.
      • Simply click on ”Send to reviewer” button to send chosen forms to be reviewed
    6. Download form.
      • Click on download form button.

      • Choose form download type, Microsoft word or pdf.
  4. Forms will have indicators next to them, it goes as follow,

  5. User can see general information about the institution by clicking on ”statistics” button, located in accreditation drop menu.

  6. File Repository is a file management.

    The user could save and keep his important files and documents or uses public documents shared on SID instead of uploading the files every time needed.

  7. Faculty will have view options on KPI`s by clicking on KPI`s button located in accreditation drop menu.

Reviewer ¶

The Reviewer part from the Accreditation module is a full flow for the reviewing process within the university, as the user will have the ability to monitor and manage the reviewers and the reviewing process as a whole.

  1. Click on Reviewer sub-module from Accreditation drop-down menu.

  2. User (Reviewer) will be able to see all assigned tasks and forms to him/her.

  3. User will have the ability to update his/her information and write down general notes and recommendations by clicking on “Update Info” button
  4. To access the assigned forms to be reviewed click on “View” button located next to Documents To Review.

  5. Access the required form/s, then fill the required information

  6. When done click on “Save”
  7. If the Reviewer is assigned to review a program, he/she will have access to these programs to write his/her recommendations

  8. Click on “Recommendations” to write a recommendation, then click on “Add Recommendation”

  9. Type down the recommendations, when done click on “Save”

  10. Reviewer will see the progress that has been made based on his/her recommendations and will have the ability to edit the recommendations as shown below

5. Curriculum Mapping ¶

The goal of this module is to have a defined set of PLOs and CLOs (Course Learning Outcomes) mapped together. This is a defined set of requirements for all courses in the university.

Benefits:This module target is to improve courses quality for theuniversity.

  1. Access this module by choosing Curriculum mapping from the Curriculum drop-down menu in the main menu.

  2. In the Curriculum mapping page, you will have a menu and a filter as shown below,

  3. To manage the CLO`s click on ”Course management” button, then click on manage next to required course.

  4. Click on learning outcomes.

  5. Click on ”manage learning outcomes” button, fill the required fields then click on ”Save”.

  6. From the learning outcomes page click on ”Mapping”.

  7. Choose the assessment method and map it by clicking on the white boxes next to them, when done click on Save.

  8. Click on ”Assessment methods” button then on ”manage assessment methods” button.

  9. Fill the required assessment method then click on ”Save”.

  10. Click on ”course sections”, then on ”learning outcome mapping” and select the required assessment method in order to add questions to be linked with the CLO`s and then to set the scale score for that question.

  11. Back to Course sections page, click on ”student assessment” button then choose the student and the required assessment method, to be able to set student score for each question, when done click on ”save”.

  12. To have a full report from the steps above click on ”reports” from Curriculum mapping page, and choose the required part to have a detailed view about it.

6. Examination ¶

Examination module is a module that is responsible of all student testing related activities including exams, Quizzes, and assignments with the ability of store and retrieve any questions needed at any time.

  1. Click on Examination located in main menu

  2. User can manage different type of examination by navigating top bar menu

  3. To start with the Exam, click on “Exam” button
  4. To create a new exam, click on “Create Exam”

  5. Fill related information to the exam, then click on “Save”

  6. Exam will appear along with its options across as shown below

    1. View button is used to see all questions in the chosen exam
    2. Manage button allows the user to edit exam information, same as shown in point number 5
    3. Design button allows the user to add and delete exam questions
    4. Delete is to delete the current exam
    5. Proctors button will allow the user to choose a proctor to manage the exam session
    6. Manage sections allows the user to choose sections which will be examined
  7. To add questions to the exam, click on design then click on Add question

  8. Choose the question and write down the question mark, then click on “Save”

  9. Repeat the same step above until all the questions has been added to the exam

    Note: Total marks for all the questions should be equivalent to the exam mark in order to proceed.

  10. Publish button will appear for the user in order to start the exam

  11. Click on “Publish” then enter exam time and date, when done click on “Publish”

  12. Once the exam starts, User should click on “Student Attendees” located in top bar menu in order to confirm students attendance for the exam

  13. Next click on “Student Attendees”

  14. Click on the check box next to the attended students
  15. Once the exam is finished the user will be able to start the correction process for the students by clicking on Correction button

  16. Click on “Check Answers” located next to the student’s name

  17. Once correction is done click on “Save”

Creating and managing a Quiz has the same steps above but there are two differences

  • the user will have two buttons to start and finish the Quiz instead of assigning a time and a date.
  • There is no attendance list of the students
  1. Click on Quiz located in the top bar menu

  2. After Creating a Quiz and adding questions to it, user can start the quiz by clicking on “Start Quiz”

  3. Next click on “Ok” to confirm the action

  4. To stop the quiz, click on “Stop Quiz”

  5. Next click on “Ok” to confirm the action

Creating and managing assignments user will use same steps in the exam but there will be no need to check the attendance for the students

  1. Click on assignments, then click on Create Assignment

  2. Fill the information then click on “Save”

  3. Manage the assignment as needed

Question Bank ¶

User can add questions to be used for examinations

  1. Click on “Question Bank”

  2. Click on “Create Question”

  3. Enter the question and fill the rest of the fields

  4. User can choose the status of the question from Question Status drop-down menu

  5. Question Status
    • Private: Only the user who created the question will be able to see the created question
    • Show On Public: all the user will be able to see and use created question
  6. If the question requires an attachment, user should click on “Assignment Question” then on “Enable Assignment” check box

  7. When done click on “Save”
  8. User will be able to edit the questions by click on “Edit” question

  9. User will have the ability link the question to a learning outcome by clicking on “Learning Outcome” button

  10. Choose the required Learning outcome, when done click on “Save”

Examination Bank ¶

  1. Click on Examination Bank

  3. Click “View” next to the required exam

7. Portfolios ¶

There are three types of portfolios in jadeer:

  • Course portfolio: this part of the module will contain information about all the courses, starting from general information about the course to the syllabus and much more.
  • Faculty portfolio: this part of the module will contain information about the faculty such as the FPP.
  • Student portfolio: this part of the module will contain information about the students within the institution.
  1. Access Portfolios module by clicking on ”Portfolios” button located in the main menu, then choose one of the three main parts of this module.

  2. To view information about a certain faculty, user can use search bar to find that faculty, then click on the faculty name.

  3. User can navigate through the information tabs available for the faculty.

  4. User can see his\her own profile by clicking on ”my portfolio” button right top of the page.

  5. Same steps apply on Student portfolio.
  6. Click on ”course portfolio”, then choose one of the available courses.

  7. Click ”edit” to manage required course.
  8. Course evaluation page shows detailed information from course evaluation survey.

  9. Click on Syllabus to view information about the syllabus for the chosen course and to edit it.
  10. Click on ”edit” for any required field to start editing process.

  11. Fill needed information, when done click on ”Save”.

  12. Manage all the tabs same ways by clicking on the required tab and edit or add any information needed.

  13. From course portfolio main page user can access settings button right top of the page.

  14. Click on ”settings”, then choose type of survey to be reflected on the charts.

  15. Click on ”save”.

  1. Click on “Faculty Performance”

  3. Choose the required form

  4. Click on “Add” to edit the chosen form

  5. User will be able to see manage and edit forms from the screen shown below

  6. Click on “Report” to get detailed reports

  7. User will be able to choose on what level the reports should be generated, and the ability to search for a specific user by using the User filter as shown below.

  8. User can edit and add recommendations to a certain faculty by clicking on “Edit” button

  9. Generate a detailed report by clicking on “HTML” button, or a summery by clicking on “Get Summery” button.
