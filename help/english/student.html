2. Dashboards ¶

This module will give the user a clear detailed information about his\her courses along with charts, it also shows tasks and surveys which are forwarded to the faculty.

  1. Access the dashboard by clicking on dashboard button from the main menu.

  2. Click on “Personal“ to access personal information, and view tasks and survey forwarded.

  3. Personal page shall appear for the user, along with tasks and surveys (if there is any).

  4. Click on “Take survey“ to fill up survey assigned.

  5. Fill up the survey, when done click on “Finish“.

  6. Manage the tasks from tasks tab, assign a task to another user by clicking on “Add new“ button.

  7. Fill up required information, when done click on “Send“ button.
  8. Click on “general“ to see general information about the institution.

3. Portfolios ¶

There are three types of portfolios in jadeer:

  • Course portfolio: this part of the module will contain information about all the courses, starting from general information about the course to the syllabus and much more.
  • Faculty portfolio: this part of the module will contain information about the faculty such as the FPP.
  • Student portfolio: this part of the module will contain information about the students within the institution.
  1. Access Portfolios module by clicking on Portfolios button located in the main menu, then choose one of the three main parts of this module.

  2. To view information about a certain faculty, user can use search bar to find that faculty, then click on the required name.

  3. User can navigate through the information tabs available for the faculty.

  4. Click on “student portfolio“ Button.
  5. Same steps above apply on Student portfolio.
  6. User can see his\her own profile by clicking on “my portfolio“ button right top of the page.

  7. Click on “Course portfolio“, then choose one of the available courses.

  8. Click “View“ on the required course.

  9. Navigate between the tabs to view required course information.

4. Examination ¶

This module will allow the students to take all the exams, assignment and quizzes directed to them

  1. Click on Examination .

  2. Click on the required type of examination, for example click on “Exam”

  3. In case there is an exam Click on “Start Exam” to start the exam
  4. Answer the questions and use “Next “and “Back “buttons to navigate between questions, to finish the exam click on “Finish Exam”

If there are any assignments to be delivered students must click on “Assignment” button to access it, same process should be applied for the Quizzes

5. Manual ¶

This module contains links and manuals needed for the user.

  1. To see all manuals, click on “Manual” located in main menu.

  2. User will see all manuals listed.
