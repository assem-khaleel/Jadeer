3. Curriculum Mapping ¶

The goal of this module is to have a defined set of PLOs and CLOs (Course Learning Outcomes) mapped together. This is a defined set of requirements for all courses in the university.

Benefits: This module target is to improving the courses quality for the university.

  1. Access this module by choosing Curriculum mapping from the Curriculum drop-down menu in the main menu.

    The upper menu contains four main buttons:
    • Program management: to manage the PLO`s within a program.
    • Course management: to manage the CLO`s within a course.
    • Reporting: Shows the Competencies Dimension.
    • Settings: Allows the user to manage Learning domains and assessment methods.
    Down to filters The user can use the filters to find required program.
  2. First to start with program management click on ”management” button.

  3. Choose the required learning domain then click on ”manage”.

  4. Fill the required fields and click on the white box to choose the outcome needed.
  5. The user can add a new learning outcome by clicking on ”Add learning outcome” button.
  6. Click on ”save”
Next component is the assessment method for the program,
  1. Click on assessment methods.
  2. First we need to add the required assessment methods by clicking on Manage assessment methods.

  3. Choose required assessment methods by checking the white boxes, then click on ”Save”.

  4. Click on ”Manage” to add Assessment component to the Assessment method.

  5. Type the Assessment component needed into the boxes and check on the required box to be activated, then click on ”Save”.

  6. Click on ”Mapping matrix” to link between learning domains and the courses, then to choose the level.

  7. Click on edit to Choose the Program level and the IPA level, then click on ”Save”.

  8. To manage the CLO`s click on ”Course management”.

  9. Click on ”learning outcomes”.

  10. Click on ”manage” learning outcomes”, fill the required fields then click on ”Save”.

  11. From the learning outcomes page click on ”Mapping” button.

     

  12. Choose the assessment method and map it by clicking on the white boxes next to them.

  13. Click on ”Assessment methods” then on manage assessment methods.

  14. Fill the required assessment method then click on”Save”.

  15. Click on ”Course sections”, then on learning outcome mapping and select the required assessment method in order to add questions to be linked with the CLO`s and then to set the scale score for that question.

  16. Back to Course sections page, click on ”Student assessment” then choose the student and the required assessment method, to be able to set the set the student score for each question, when done click on ”Save”.

  17. To have a full report from the steps above click on ”reports” from Curriculum mapping page, and choose the required part to have a detailed view about it.

  18. By clicking on the Settings from Curriculum mapping page the user will be able to modify the assessment methods along with the learning domains, by using edit and delete buttons.

4. Program Tree ¶

This module allows the user to link between the university mission all the way down to the program objectives in an organized way, to be shown in a tree like chart showing the whole linking process.

  1. Access this module by choosing Program Tree from Curriculum drop-down menu in the main menu.

  2.  Choose a program from the list or use the search filters to find the required program, then click on ”Edit Tree”.

  3. In the university mission page type in the needed keywords into the box, then click on ”save”.

  4. The university keywords will be linked with keywords from the college mission.
  5. Click on ”Consistency between university mission to college mission” button.

  6. Choose college mission keywords from the college mission, then link between the university mission keywords and the college mission keywords by clicking on the mapping boxes.

  7. Click on ”Consistency between college mission to program mission” button.

  8. Choose college mission keywords from the college mission, then link between the university mission keywords and the program mission keywords by clicking on the mapping boxes, then click on “Save”.

  9. Click on ”program mission to program objectives” button.

  10. Link between program mission and program objectives by clicking on the mapping boxes.

  11. Click on ”PLO to program objectives” button.

  12. Link between PLO`s and program objectives by clicking on the mapping boxes, then click on save

  13. Now to view the program tree that has been created click on ”View tree”.

5. Examination ¶

Examination module is a module that is responsible of all student testing related activities including exams, Quizzes, and assignments with the ability to store and retrieve any questions needed at any time.

  1. Click on Examination located in main menu

  2. User can manage different type of examination by navigating top bar menu

  3. To start with the Exam, click on “Exam” button
  4. To create a new exam, click on “Create Exam”

  5. Fill related information to the exam, then click on “Save”

  6. Exam will appear along with its options across as shown below

    • View button is used to see all questions in the chosen exam
    • Manage button allows the user to edit exam information, same as shown in point number 5
    • Design button allows the user to add and delete exam questions
    • Delete is to delete the current exam
    • Proctors button will allow the user to choose a proctor to manage the exam session
    • Manage sections allows the user to choose sections which will be examined
  7. To add questions to the exam, click on design then click on Add question

  8. Choose the question and write down the question mark, then click on “Save”

  9. Repeat the same step above until all the questions has been added to the exam

    Note: Total marks for all the questions should be equivalent to the exam mark in order to proceed.

  10. Publish button will appear for the user in order to start the exam

  11. Click on “Publish” then enter exam time and date, when done click on “Publish”

  12. Once the exam starts, User should click on “Student Attendees” located in top bar menu in order to confirm students attendance for the exam

  13. Next click on “Student Attendees”

  14. Click on the check box next to the attended students
  15. Once the exam is finished the user will be able to start the correction process for the students by clicking on Correction button

  16. Click on “Check Answers” located next to the student’s name

  17. Once correction is done click on “Save”

    s

Creating and managing a Quiz has the same steps above but there are two differences

  • the user will have two buttons to start and finish the Quiz instead of assigning a time and a date.
  • here is no attendance list of the students
  1. Click on Quiz located in the top bar menu

  2. After Creating a Quiz and adding questions to it, user can start the quiz by clicking on “Start Quiz”

  3. Next click on “Ok” to confirm the action

  4. To stop the quiz, click on “Stop Quiz”

  5. Next click on “Ok” to confirm the action

Creating and managing assignments user will use same steps in the exam but there will be no need to check the attendance for the students

  1. Click on assignments, then click on Create Assignment

  2. Fill the information then click on “Save”

  3. Manage the assignment as needed

6. Policies and Management ¶

This module will allow the user to see and organize all policies and procedures within the institution, and managing all the meeting with the ability of keeping all the information needed from the meeting for any future use.

A. Committee

Committee sub-module will allow the user to create Committees for any upcoming meetings.

  1. From Policies and Management drop down menu click on Committee

  2. To add a new Committee, click on “Add New”

  3. Enter the required information related to the committee, then add Committee members by clicking on “Add More” button, Choose the Group leader by checking the white box across the user, when done click on “Save”

  4. User will have the ability to Edit, Delete, View or Download any committee by clicking on the needed command across desired committee

B. Room Management

In this sub module the user will have the ability to manage meeting rooms and enter all the required information related to the rooms and equipment’s needed for the meetings.

  1. From Policies and Management module tab Click on “Room Management”

  2. To Create new meeting room Click on “Create Room”

  3. Enter the required information then click on “Save”.

  4. To manage the equipment’s needed for the meetings click on “Equipments”
  5. To add equipment’s Click on Create “Equipments”

  6. Enter required information then click on “Save”

  7. List of all the equipment’s will be available with the ability to edit and delete it.

  8. Back to the room management list of all the rooms created will be shown, user will have the ability to edit or delete any room, or manage room equipment’s by clicking on “Manage”

  9. After clicking on “Manage” Choose the needed equipment’s by checking the white boxes across the desired equipment

  10. When done click on “Save”

  12. First Click on Add Topic

7. Accreditation ¶

Through this module the user will be able to control the accreditation process for both national and international, programmatic and institutional Accreditation.

  1. First step will be creating accreditation, by clicking on Accreditation drop-down menu, then choose the accreditation type.

  2. From the national accreditation page click on ”Create Accreditation”.

  3. Choose the type of accreditation forms to be generated, then click on ”Next” button.

  4. Set a due date by clicking on ”due date”.

  5. Choose the required due date then click on ”Save Changes”.

  6. To access the forms, click on ”manage”.

  7. There are function buttons to help the user managing the forms.
       
    1. Set a due date, same way as setting a due date for the whole forms.
    2. Assign a user to fill or manage the forms.
      • Click on the add user button, then choose what type of user you want to add (Assessor or Reviewer).

      • find the required user by searching the users name, select the users name then click on ”Add”.

    3. Edit forms button.
      • Click on edit forms button, then fill up or edit the form as needed, when done click ”Save Changes” to save the form without marking it as finished (to be finished later), or click on ”Save and finish” to save the form and mark it as finished.

      • to retrieve a previous history for the form, click on the history button top right of the form

      • the user will be able to see both forms, the current form and the chosen form from the history, every change that has been made on the form will be reflected as a log along with the date and time.

      • Change or keep the needed information, then click on ”Save Changes”.
    4. View the form.
      • the user will be able to view the current form by clicking on view form button.

    5. Send to reviewer.
      • simply click on the form to send it to be reviewed.
    6. Download form.
      • Click on download form button.

      • Choose form download type, Microsoft word or pdf.
  8. Forms will have indicators next to them, it goes as follow,

  9. User can see general information about the institution by clicking on ”Statistics” button, located in accreditation drop menu.

  10. Push to AIMS is a function that will allow the user to send all the forms to NCAAA directly, to send the forms first click on ”Push to AIMS” button.

  11. Choose the Accreditation type, institutional or programmatic.
  12. Choose the required forms, then click on Push to ”AIMS”.

  13. File Repository is a file management.

    The user could save and keep his important files and documents or uses public documents shared on SID instead of uploading the files every time needed.

12. Survey ¶

This module allows the user to create all types of surveys to fulfill the institution needs, surveys can be created and saved to be used every semester such as Course evaluation survey, or it can be a customized survey that achieves a certain goal.

  1. Access survey module by clicking on survey located in main menu.

  2. Then to start off user should create a survey, click on ”Create survey” to start this process.

  3. Fill required information, then choose user type for the survey, when done click ”save” and go next button to design the survey or click on save and continue later to exit without designing the survey, user also can click on copy an exciting survey check box to choose previously designed survey.

  4. Start designing the survey by using following buttons:
    • Add page: to add a page to the survey, user can choose either to add it before or after the current page.
    • Add questions: add questions to the survey.
    • Copy: copy current survey page.
    • Edit: edit current survey page.

  5. Click on ”Add questions” to add a question to the survey.
  6. Fill required question fields and choose question type, if it’s a multiple question enter the answers under questions choices tab, user can also choose if its required to answer the current question by clicking on the white check box at the bottom of the page, when done click on ”Save”.

  7. Question will appear along with its choices, user can click on ”Copy” button to copy the question or ”move” button to move it up or down, once user is done from question designing process click on ”Finish”.

  8. To send the survey click on ”Evaluations” button.

  9. There are many useful functions in this page as:
    • Results: shows user the results of this survey.
    • Remind: send notification to all the users to remind them to fill up the survey.
    • Preview: check student’s answers.
    • Edit and resend: edit the survey and send it back again to users.
    • Delete: delete current evaluation.

  10. Click on ”Add” to send the survey to users, then fill required information then click on ”Send” button.

  11. Once done click on ”Results” from survey landing page to see survey results.

13. Alumni Center ¶

This module will allow the institution to stay in touch with its alumni and to connect with the employers of their students, this process will insure learning quality within the institution.

  1. Access Alumni Center module by clicking on Alumni center from survey drop down menu.

  2. To add new alumni, click on ”Create alumni” button.

  3. Fill required information for the Alumni which will be stored in jadeer.

  4. When done click on ”Save” button.

  7. To add an employer, click on ”Employer” button, then click on ”Create employer” button.

  8. Fill the required information then click on ”Save”.

Note: above is a manual process for adding both Alumni and employers, although if the institution have this information about Alumni and employers it will be automatically integrated with jadeer.

14. Portfolios ¶

There are three types of portfolios in jadeer:

  • Course portfolio: this part of the module will contain information about all the courses, starting from general information about the course to the syllabus and much more.
  • Faculty portfolio: this part of the module will contain information about the faculty such as the FPP.
  • Student portfolio: this part of the module will contain information about the students within the institution.
  1. Access Portfolios module by clicking on Portfolios button located in the main menu, then choose one of the three main parts of this module.

  2. To view information about a certain faculty, user can use search bar to find that faculty, then click on the required name.

  3. User can navigate through the information tabs available for the faculty.

  4. Same step applies on the student portfolio.
